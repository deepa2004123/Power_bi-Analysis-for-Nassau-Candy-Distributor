# Power_bi-Analysis-for-Nassau-Candy-Distributor

## Project Overview
This Power BI project analyzes shipping performance and route efficiency for the Nassau Candy Distributor.  
The dashboard helps identify delivery delays, inefficient routes, and shipping mode performance to support data-driven logistics decisions.

---

## Key Objectives
- Analyze average shipping lead time  
- Identify delay frequency using a configurable lead-time threshold  
- Compare performance across shipping modes  
- Evaluate route efficiency by region and state  
- Enable drill-down for route-level and order-level analysis  

---

## Key KPIs
- Average Lead Time (Days)  
- Total Shipments  
- Delay Frequency (%)  
- Route Efficiency Score  

---

## Dashboard Features
- Date range slicer (Year / Month)  
- Region and State filters  
- Ship Mode comparison  
- Dynamic Lead Time Threshold (What-If parameter)  
- Top and Bottom 10 slowest shipping routes  
- Geographic shipment distribution  

---

## Tools and Technologies
- Power BI Desktop  
- DAX (Measures, Calculated Tables, What-If Parameters)  
- Data Modeling and Visualization  
- GitHub for project hosting
